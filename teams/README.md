# Team Workspaces

Each team works in its own `team-N/` folder. Use the subfolders as follows:

- `mermaid/` — Mermaid diagram source (`.md` or `.mmd`).
- `dsl/` — Structurizr DSL models (`.dsl`).
- `adrs/` — Architecture Decision Records, numbered from `0001`.
- `capstone/` — your Day 3 capstone artifacts.

Commit with the message format `team-N: what changed` (e.g. `team-3: add container diagram`).
