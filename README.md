# Mastering the C4 Model for Software Architecture

A three-day, hands-on course for learning the C4 model using a fictional healthcare system, MedTrack.

## What's in this repo

| Folder | When it's used |
| --- | --- |
| `briefs/` | Exercise inputs — the MedTrack brief, NFR hints, and environment cards |
| `cheatsheets/` | Reference — Mermaid C4, Structurizr DSL, review checklist, AI prompt patterns |
| `templates/` | Day 3 — ADR templates (Nygard and MADR) plus a worked exemplar |
| `dsl/` | Day 2 — reference and starter Structurizr DSL models |
| `sample-code/` | Day 2 demo — a small Spring Boot app to diagram from code |
| `teams/` | Your team's workspace for diagrams, DSL, ADRs, and capstone artifacts |

## Setup

Nothing to install. All class tools are browser-based:

- [mermaid.live](https://mermaid.live) — Mermaid C4 diagrams
- [structurizr.com/dsl](https://structurizr.com/dsl) — Structurizr DSL playground
- [app.diagrams.net](https://app.diagrams.net) — draw.io with C4 shapes

Git and VS Code with GitHub Copilot Chat are optional but recommended — you'll use them for the code-to-diagram demo and to commit your team's work.
